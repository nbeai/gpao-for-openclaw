# BEAI Runtime v0.6.20 릴리스 노트

v0.6.20은 v0.6.19 위에 Friction-Aware Gate를 추가한 런타임 판단 패치입니다.

핵심 변화:

- `fast_lane`: 초안, 사고 정리, 읽기 전용 분석, 후보안 작성은 사용자 흐름을 끊지 않고 빠르게 처리합니다.
- `quiet_check`: 자료 기반 요약, 내부 검토, 파일/문서 후보 작업은 가능한 한 조용히 확인하고 결과에 확인/미확인 경계만 짧게 남깁니다.
- `approval_gate`: 외부 발송, 공개 게시, 삭제/덮어쓰기, 권한/계정 변경, cron 등록, 장기 기억 승격, 결제/계약/법률/세무성 판단은 실행 전에 사용자 승인을 요구합니다.
- `post_action_verify`: 코드 수정, 빌드, 테스트, 배포파일, 패키지 버전 정합성, Telegram 전달처럼 실행 뒤 증거가 필요한 작업은 검증 전 완료로 닫지 않습니다.
- `do_not_automate`: 고책임 영역과 민감정보, 되돌리기 어려운 외부 영향 작업은 자동화 대상으로 승격하지 않습니다.

제품 기준:

- BEAI는 브레이크가 아니라 클러치처럼 작동해야 합니다.
- 대부분의 저위험 작업은 빠르게 흐르게 두고, 책임이 사용자 밖으로 넘어가거나 되돌리기 어려운 순간에만 마찰을 올립니다.
- 검증은 가능하면 조용히 수행하고, 사용자에게는 “확인한 것 / 아직 미확인인 것 / 승인 필요한 지점”만 남깁니다.

경계:

- 이 버전은 런타임 판단과 패키지 계약을 강화합니다.
- 이 버전 자체가 skill, agent, cron, memory, external send, public publish, Gateway restart, live reinstall을 자동 실행하지 않습니다.
